Visual Programmer Version History

1.2.336
  • Added ability to specify working directory upon startup, and have that preference optionally remembered per user (actually, per device, so Hummingbird and Finch prefs are stored separately).
  • Added "Visual Programmer Files" section to the Settings tab. This is where you specify whether you want the app to remember the directory used for your files.
  • Added About section to the Settings tab
  • Improved audio file installation, and now includes a progress bar
  • Anonymous stats are now sent to the artsandbots.com server during update check. This helps us better understand the variety of computer environments of our users (operating system, Java version, etc.)
  • Bug fixes and performance improvements
1.1.318
  • Now distributing as a standalone app for Mac, Windows, and Linux. Zip file version still available.
  • Retired Java Web Start distribution. Still available for legacy users, but will not be updated anymore.
  • Added Settings tab
  • Added update checking
  • Made the app more resistant to hard crashes on the Mac when you simply yank out the USB cord
  • Improved Installation and Connection Guide
  • Bug fixes and performance improvements
